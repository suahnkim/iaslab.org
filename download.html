SNS image compression

Following dataset is useful for testing new image compression techniques that are specifically targeting SNS platform. Currently, we offer images that are uploaded and downloaded from Instagram SNS. We hope to expand this work in the future.

• Instagram compression test dataset: This contains resized and unenhanced jpeg images. All images are resized to an optimal size for Instagram post (1080p x 566p ~ 1350p) before uploading it to Instagram. Since an Instagram post can have multiple number of images, dataset is organized by number of images in a post. Each dataset contains the uploaded original version, downloaded version, and an image dataset description file.

SNS built-in enhancement

Following dataset is useful for testing the effect of built-in SNS image enhancement.

• Instagram filter dataset: This dataset contains resized and filter-applied JPEG images. All images are resized to an optimal size for Instagram post (1080p x 566p ~ 1350p) before uploading it to Instagram. The dataset contains images enhanced with 41 image filters (1977, Aden, Amar, Ashb, Bran, Broo, Char, Clar, Crem, Dogp, Earl, Ging, Ginz, Hefe, Hele, Huds, Inkw, Juno, Kelv, Lark, Lo-F, Ludw, Mave, Mayf, Moon, Nash, nfit, Perp, Reye, Rise, Sier, Skyl, Slum, Stin, Sutr, Toas, Vale, Vesp, Wald, Will, X-Pr). Since an Instagram post can have multiple number of images, dataset is organized by number of images in a post. Each dataset contains the uploaded original version, downloaded version, and an image dataset description file.

     ⚬ 1 image per post (20 posts for each filter):
      [Color JPEG], [Grayscale JPEG], [Cite]
     ⚬ 2 images per post (10 posts for each filter):
      [Color JPEG], [Grayscale JPEG], [Cite]
     ⚬ 3 images per post (10 posts for each filter):
      [Color JPEG], [Grayscale JPEG], [Cite]
     ⚬ 6 images per post (10 posts for each filter):
      [Color JPEG], [Grayscale JPEG], [Cite]
     ⚬ All Instagram filter datasets:
      [Color JPEG], [Grayscale JPEG], [Cite]
