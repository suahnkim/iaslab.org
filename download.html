SNS image compression dataset

This dataset is useful for testing new image compression techniques that are specifically targeting SNS platform. Currently, we offer images that are uploaded and downloaded from Instagram SNS. We hope to expand this work in the future.

• Instagram compression test dataset: This contains resized and unenhanced jpeg images. All images are resized to an optimal size for Instagram post (1080p x 566p ~ 1350p) before uploading it to Instagram. Since an Instagram post can have multiple number of images, dataset is organized by number of images in a post. Each dataset contains the uploaded version, downloaded version, and an image dataset description file.
     ⚬ 20 posts with 1 image per post:    [40 color JPEG dataset],   [40 grayscaled JPEG dataset],   [40 color HEIC/HEIF dataset]
     ⚬ 10 posts with 2 images per post:   [40 color JPEG dataset],   [40 grayscaled JPEG dataset]
     ⚬ 10 posts with 3 images per post:   [60 color JPEG dataset],   [60 grayscaled JPEG dataset]
     ⚬ 10 posts with 4 images per post:   [80 color JPEG dataset],   [80 grayscaled JPEG dataset]
     ⚬ 10 posts with 5 images per post:   [100 color JPEG dataset],   [100 grayscaled JPEG dataset]
     ⚬ 10 posts with 6 images per post:   [120 color JPEG dataset],   [120 grayscaled JPEG dataset]
     ⚬ All Instagram compression datasets:   [Color JPEG dataset],   [Grayscaled JPEG dataset]

SNS built-in enhancement dataset

This dataset is useful for testing the effect of built-in SNS image enhancement.

• Instagram filter dataset: This dataset contains resized and filter-applied JPEG images. All images are resized to an optimal size for Instagram post (1080p x 566p ~ 1350p) before uploading it to Instagram. The dataset contains images enhanced with 41 image filters (1977, Aden, Amar, Ashb, Bran, Broo, Char, Clar, Crem, Dogp, Earl, Ging, Ginz, Hefe, Hele, Huds, Inkw, Juno, Kelv, Lark, Lo-F, Ludw, Mave, Mayf, Moon, Nash, nfit, Perp, Reye, Rise, Sier, Skyl, Slum, Stin, Sutr, Toas, Vale, Vesp, Wald, Will, X-Pr). Since an Instagram post can have multiple number of images, dataset is organized by number of images in a post. Each dataset contains the uploaded version, downloaded version, and an image dataset description file.
     ⚬ 41x20 posts with 1 image per post:    [860 color JPEG dataset],   [860 grayscaled JPEG dataset]
     ⚬ 41x10 posts with 2 images per post:   [860 color JPEG dataset],   [860 grayscaled JPEG dataset]
     ⚬ 41x10 posts with 3 images per post:   [1,260 color JPEG dataset],   [1,260 grayscaled JPEG dataset]
     ⚬ 41x10 posts with 6 images per post:   [2,520 color JPEG dataset],   [2,520 grayscaled JPEG dataset]
     ⚬ All Instagram filter datasets:   [Color JPEG dataset],   [Grayscaled JPEG dataset]
